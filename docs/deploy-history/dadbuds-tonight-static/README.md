# DadBuds Static Deploy History

This folder preserves the old Metropolis-hosted DadBuds static deploy pack and
zip history that previously powered `dadbuds.lol/join`.

The active DadBuds landing and intake flow now lives in the React app:

- `/join` uses the same rotating public landing page as `/`.
- `/join/signup` carries the short Spokane beta intake.
- `/join/thanks` is the post-capture confirmation route.

Keep this archive for comparison, rollback reference, and Netlify Drop history.
Do not treat it as the active source for new landing-page edits.
